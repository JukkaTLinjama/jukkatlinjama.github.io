Vacuum Sample Lab v1.6

This version focuses on loop extraction and behaviour testing: period estimation, snapping the end to integer motor periods, fixed-length outside-handle crossfade, output gain, motor/air layer RMS comparison, and two softened decorrelated RMS-normalized velvet air layers. Space toggles start/stop. Orange slider markers show effective post-load layer values.

Sample input

File input is the first path. Later this can also load assets/test-loop.wav from GitHub Pages.

Loop recording / download

Mic recording needs HTTPS or localhost. Use this rarely; normal work can use file input and clipped WAV export.
