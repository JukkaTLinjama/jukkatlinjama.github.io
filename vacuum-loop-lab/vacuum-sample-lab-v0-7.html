          Vacuum Sample Lab v0.7


          This lab tests one morphing tonal vacuum motor loop plus synthetic air layers.
          Keep the tonal motor as a single morphing source to avoid beating between similar loops.
          Load increases sample playbackRate from 1.00 to 1.50, raises motor gain up to +3 dB,
          lowers synthetic air by about -8 dB, and raises a subtle harmonic/noise layer.
          


          The clip editor selects a raw region, estimates the motor period from the selected clip using autocorrelation,
          and can snap the loop end to an integer number of periods from the selected start.
          Loop crossfade mixes the selected end into the selected beginning instead of fading both edges to zero.
          Output gain is applied to the generated clip/export buffer, so it affects downloaded WAV level
          and the loop used by the motor source.
          


          Current limitations: no waveform display, no LUFS standard measurement, no automatic best-loop search,
          and no IndexedDB storage. The current meters are RMS/peak only; layer RMS is an approximate energy comparison, not a tonality detector.
